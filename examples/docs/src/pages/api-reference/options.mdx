# Plugin Options

The default plugin options are

```js
{
  defaultLayouts: {},
  extensions: [".mdx"],
  mediaTypes: ["text/markdown", "text/x-markdown"],
  rehypePlugins: [],
  remarkPlugins: [],
  gatsbyRemarkPlugins: [],
  globalScope: `export default {}`
}
```
